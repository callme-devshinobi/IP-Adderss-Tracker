A responsive IP Address Tracker application that allows users to search for any IP address or domain and view its geolocation details in real time.

This project was completed as part of a challenge from **Frontend Mentor**.

## 🚀 Features

- 🔎 Search for any IP address or domain  
- 📍 Displays:
  - IP Address  
  - Location (City, Region, Country)  
  - Timezone  
  - ISP  
- 🗺 Interactive map powered by Leaflet.js  
- 📱 Fully responsive design  
- ⚡ Real-time data fetching using API
